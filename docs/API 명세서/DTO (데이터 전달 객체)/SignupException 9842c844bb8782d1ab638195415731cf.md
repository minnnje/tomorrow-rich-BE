# SignupException

API 분류 태그: API Exception
태그: [인증] 회원가입

1. 비밀번호 불일치

```json
{
  "timestamp": "시간",
  "statusCode": 400,
  "error": "비밀번호 불일치",
  "message": "400 BAD_REQUEST \"비밀번호가 일치하지 않습니다.\"",
  "details": "uri=/api/auth/signup",
  "errorCode": "PASSWORD_MISMATCH"
}
```

1. 아이디 중복

```json
{
  "timestamp": "시간",
  "statusCode": 409,
  "error": "이미 사용 중인 아이디",
  "message": "409 CONFLICT \"이미 사용 중인 아이디입니다.\"",
  "details": "uri=/api/auth/signup",
  "errorCode": "DUPLICATED_LOGIN_ID"
}
```
